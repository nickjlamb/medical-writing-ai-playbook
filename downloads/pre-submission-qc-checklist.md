Final-gate quality control for AI-assisted manuscripts, abstracts, and regulatory submissions. Run on the final, submission-ready version — not on a working draft.

## Reference integrity

- [ ] Every cited reference exists; DOI or PMID resolves
- [ ] Claims supported by cited source (closed-loop verification reviewed and accepted)
- [ ] No "borrowed" findings — claims attributed to the correct study, population, analysis type
- [ ] Subgroup, post-hoc, exploratory qualifiers preserved
- [ ] Self-citation appropriate and not over-used

## Data and statistical accuracy

- [ ] Numbers match source exactly (p-values, hazard ratios, percentages, sample sizes)
- [ ] Confidence intervals reported where the source reports them
- [ ] No impossible statistics (GRIM violations, p-value/test statistic inconsistencies)
- [ ] ITT vs. mITT vs. per-protocol correctly labelled
- [ ] Endpoints correctly identified as primary, secondary, exploratory

## Image and figure integrity

- [ ] No image duplications across this submission or prior published work
- [ ] No splicing, cropping, contrast adjustments that could be challenged
- [ ] Image-integrity check run (e.g., Imagetwin, Proofig)
- [ ] AI-generated images explicitly labelled as conceptual
- [ ] Figure captions match figure content and underlying source

## AI use and disclosure

- [ ] AI disclosure language drafted using the Disclosure Language Template
- [ ] Disclosure specifies tool name, version, scope, named reviewer
- [ ] Disclosure placed where the journal/venue requires it
- [ ] Audit-trail log complete and accessible
- [ ] No undisclosed AI use

## Methodology and reporting compliance

- [ ] Reporting guideline followed (CONSORT, PRISMA, STROBE, etc.)
- [ ] Methods section complete enough for reproducibility
- [ ] Pre-registration referenced where applicable
- [ ] Limitations clearly stated

## Plagiarism and originality

- [ ] iThenticate or equivalent similarity check run
- [ ] No verbatim text reuse without proper attribution
- [ ] Self-plagiarism (text reuse from author's prior work) flagged where significant
- [ ] Quotation marks and citation used correctly for directly reproduced passages

## Authorship and affiliations

- [ ] All authors meet authorship criteria (ICMJE or journal-specific)
- [ ] No "courtesy" or "gift" authorship
- [ ] Each author's affiliation current and accurate
- [ ] ORCID iDs included for each author
- [ ] Conflicts of interest disclosed in full
- [ ] AI is NOT listed as an author

## Venue-specific checks

- [ ] Journal's specific AI-use policy reviewed
- [ ] Word, figure, and table limits respected
- [ ] Required sections present (data availability, ethics approval, funding)
- [ ] File formats and naming conventions match submission system requirements
- [ ] Cover letter aligns with submitted content
