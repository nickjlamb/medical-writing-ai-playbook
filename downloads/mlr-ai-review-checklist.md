# MLR-with-AI Review Checklist

A structured checklist for MLR reviewers working on AI-assisted promotional or scientific content. Extends a standard MLR checklist with AI-specific items; does not replace it.

## Before the review

- [ ] AI audit-trail log is complete and accessible
- [ ] All cited references are accessible as full text
- [ ] Approved messaging framework / claims matrix is to hand
- [ ] Applicable code identified (ABPI, IFPMA, FDA OPDP, etc.)
- [ ] Risk tier of the deliverable documented
- [ ] Disclosure language for the deliverable drafted

## AI-specific checks

### Reference and source integrity

- [ ] No fabricated references — every reference exists, DOI/PMID resolves, supports the claim
- [ ] No fabricated data — numbers match source data exactly (≥20% spot-checked)
- [ ] No merged findings from different studies, populations, or analysis types
- [ ] No drifted qualifiers (subgroup, post-hoc, exploratory, secondary endpoint)

### Claim integrity

- [ ] Claims fall within the approved messaging framework
- [ ] No off-label implications or scope expansion
- [ ] Comparative claims substantiated by head-to-head data
- [ ] Language strength matches evidence strength

### Safety and balance

- [ ] Fair balance maintained
- [ ] Safety claims verified with the same rigour as efficacy claims
- [ ] Limitations of the evidence visible, not minimised

### Compliance and documentation

- [ ] Code-specific signals checked (e.g., via MedCheckr)
- [ ] Disclosure language present and venue-appropriate
- [ ] AI use consistent with sponsor SOP
- [ ] Audit log is contemporaneous (dated alongside the work)

### AI-generated visuals

- [ ] AI-generated figures not presented as primary scientific data
- [ ] Patient depictions don't imply outcomes
- [ ] No identifiable real people or recognisable products without licence

## Standard MLR checks (with AI lens)

- [ ] All claims substantiated against cited evidence
- [ ] References numbered, formatted, and accessible
- [ ] Prescribing information present and current
- [ ] Approval workflow followed
